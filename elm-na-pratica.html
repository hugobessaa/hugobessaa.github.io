Elm na prática

Aprenda a construir webapps simples e confiáveis com a incrível linguagem de programação Elm.

Comece grátis ⚡️
Por Hugo Bessa 👨🏻‍💻

Elm traz confiabilidade e simplicidade para o desenvolvimento front-end. Webapps estão cada vez mais interativos e complexos. Aprenda, neste curso, a criar webapps modernas utilizando uma linguagem de programação funcional que mudará completamente a forma como você pensa sobre desenvolvimento.

Construir webapps não precisa ser difícil 😄

  • 📈 Abordagem incremental

    Partindo de conceitos simples, mergulharemos na linguagem de programação Elm para construir um webapp do zero.

  • ✏️ 100% prático

    Cada módulo foi desenvolvido para você aprender com a mão na massa. Desenvolveremos um projeto para que você possa tornar real as suas próprias ideias.

  • 🗺 Do começo ao fim

    Do primeiro texto à publicação do seu webapp, Elm na prática te levará em uma experiência guiada para aprender a desenvolver webapps.
