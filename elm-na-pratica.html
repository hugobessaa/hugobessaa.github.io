Elm na prática

Construa webapps simples e confiáveis com a incrível linguagem de programação Elm.

Elm traz confiabilidade e simplicidade para o desenvolvimento front-end. Webapps estão cada vez mais interativos e complexos. Aprenda, neste curso, a criar webapps modernas utilizando uma linguagem de programação funcional que mudará completamente a forma como você pensa sobre desenvolvimento.

Construir webapps pode ser fácil

Partindo de conceitos simples, mergulharemos na linguagem de programação Elm para construir um webapp. Em cada vídeo aprenderemos conceitos novos com a mão na massa. No final, teremos um projeto completo, pronto para ser publicado!

Conteúdo

Durante o curso construiremos um webapp para contar pontos, que pode ser utilizado em jogos de tabuleiro ou de cartas. Este é um projeto que cobrirá os principais conceitos da linguagem de programação Elm.
